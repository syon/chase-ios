# chase-ios

## dev

```bash
$ cd Chase
$ react-native run-ios
```

## package upgrade

```bash
$ brew update

$ brew doctor

$ brew upgrade yarn

$ rm -rf node_modules/

$ yarn

$ react-native link

$ yarn upgrade-interactive
```

## Re:Install Xcode Project

```bash
$ react-native init Chase

$ cd Chase/

$ react-native run-ios

# Restore package.json

$ rm -rf node_modules/

$ yarn

$ react-native link
```
